Barbecue (Шашлыки): bbq-tomorrow.ru / github - веб-приложение для организации совместных мероприятий. Обладает достаточно большой функциональностью: Каждый пользователь после регистрациии может стать участником системы, после чего может создавать событие - открытое или приватное, с указанием адреса и времени его начала. Каждый участник может присоединиться к открытому событию, и , при наличии у него пин-кода, - к приватному. Пользователю доступны также - точка сбора (отображается на карте), добавление фотографий и комментариев к событиям, получение инормации по электронной почте об обновлении событий. Изначально было развернуто на Amazon EC2, недавно переехало на vps от REG.RU
Askme (TheQuestioned): thequestioned.herokuapp.com / github - веб-приложение для обмена вопросами и ответами.(Для загрузки нужно подождать из-за особенности хостинга). В приложении реализованы: регистрация пользователя, возможность задать вопрос зарегистрированным пользователям, добавление хэштегов, редактирование профиля, защита от спама с помощью капчи
Marketplace-API: Marketplace-API (github) - Rest API приложение, выполненное по книге «API on Rails 6» от Alexandre Rousseau. Целью автора книги являлось предоставление в ней методологии разработки RESTful API, c учётом лучших практик.
